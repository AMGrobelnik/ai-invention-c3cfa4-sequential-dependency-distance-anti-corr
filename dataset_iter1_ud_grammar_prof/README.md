## Summary

Per-treebank grammar profiles extracted from all 336 Universal Dependencies treebanks (commul/universal_dependencies on HuggingFace). Each of the 336 examples represents one UD treebank and contains: head-direction profiles per deprel, sibling-ordering templates, structural statistics (tree depth, branching factor, projectivity, sentence length), per-deprel frequency counts, case feature proportions, and typological metadata from WALS Chapter 49A. Covers 31 language families, 323 written + 11 spoken + 2 sign language treebanks, with 126/336 WALS 49A matches. Schema-validated against exp_sel_data_out.json.
